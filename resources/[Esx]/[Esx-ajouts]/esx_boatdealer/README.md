# esx_airplanedealer
## How to install
• Import `esx_aerodealer.sql` to your database<br>
• Donwload the script and drop it to your `ressources/` folder<br>
• Add `ensure esx_airplanedealer` to your `server.cfg`<br>
• Configure config.lua<br>
• Enjoy !<br>
